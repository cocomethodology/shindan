# 2026-07-17（金）| 愛の受け取り方のクセ

## 質問投稿

```
「そんなに大事にしてくれなくていいのに」——大切にされた瞬間、なぜかそう言ってしまう。

その一言に、愛の受け取り方のクセが透けて見える。

自分に一番近いのはどれ？

A〜D、コメントへ。いくつ当てはまった？

A. 大切にしてもらうと、まず「ごめんね、悪いよ」と謝る言葉が先に出てしまう
B. 優しくされるほど、「これ、何かのお返しを期待されてるのかな」と身構えてしまう
C. 素直に「ありがとう、嬉しい」と、受け取った気持ちをそのまま言葉にできる
D. 大事にされた瞬間はすごく嬉しいのに、後になってから急に照れくさくなって離れたくなる
```

## 返信欄

```
「そんなに大事にしてくれなくていいのに」

5万人の本音を聴いてきて、何度も出会った瞬間がある。
大切にされた時に出てくる最初の一言に、その人の愛の受け取り方が透けて見える。

──────────────
A｜大切にされると、まず「ごめんね」が先に出る

受け取ることに、まだ慣れていないだけだ。
謝罪で受け止める癖は、悪いことじゃない。ただ、それは「ありがとう」に置き換えられる。

──────────────
B｜優しくされるほど「お返しを期待されてる？」と身構える

これは、過去にどこかで、優しさの裏に条件があった経験があるということ。
警戒は、そこで自分を守ってくれた大切な力だった。ただし、警戒したままでいると、本当に無条件で差し出された優しさまで疑ってしまい、大切にされる機会そのものを自分から遠ざけてしまうことがある。

──────────────
C｜素直に「ありがとう、嬉しい」と受け取った気持ちを言葉にできる

これができる人は、実はそう多くない。
「ごめんなさい」より「ありがとう」が出てくる瞬間は、それ自体が答えだ。

──────────────
D｜嬉しかったのに、後から急に照れくさくなって離れたくなる

感じた喜びが大きいほど、処理しきれずに一度距離を取りたくなる。
これも、ちゃんと受け取っている証拠。ただ、消化の仕方が少し不器用なだけ。

──────────────

謝る受け取り方も、身構える受け取り方も、照れて離れる受け取り方も、どれも間違いじゃない。
ただ、練習すれば、少しずつ変えていける。

感情はある。依存はしない。
「ありがとう」と堂々と受け取れる人は、下手に出た側として見下されない。素直に受け取れる分だけ、相手はもっと大切にしたくなる。

じゃあ、「ありがとう」がまだうまく言えない自分は、どこから練習すればいいのか。その一歩目を、会員の方に置いてる。
```
