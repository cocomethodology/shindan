# 2026-07-18（土）| 又聞きとの距離の取り方

## 質問投稿

```
「〇〇さんが、あなたのことこう言ってたよ」——本人からじゃない言葉を、風のうわさで聞いた瞬間があった。

直接言われたわけじゃない。でも、心には確かに刺さる。

そんな時、自分に一番近いのはどれ？

A〜Dで教えて。

A. 「本人に直接言ってくれればいいのに」ともやもやしながら、結局何も言わずその場をやり過ごす
B. 気になって、話をした人の周りの反応を、それとなく探ってしまう
C. 黙っていられず、噂の出どころまで直接確かめに行ってしまう
D. 又聞きした瞬間は動揺しても、時間が経てば「本人の言葉じゃない」と自然に距離を置ける
```

## 返信欄

```
「〇〇さんが、あなたのことこう言ってたよ」

本人からじゃない言葉ほど、なぜか深く刺さる。直接言われた言葉より、ずっと防ぎようがない。

──────────────
A｜もやもやしながら、結局何も言わずやり過ごす

我慢強さでもある。ただ、黙ってやり過ごすほど、同じことがまた起きた時に「今回もまた黙るしかない」という前例を自分の中に作ってしまう。何も言わないことと、何も感じていないことは別だと、自分にだけは正直でいい。

──────────────
B｜話をした人の周りの反応を、それとなく探ってしまう

情報を集めたい気持ちは自然なことだ。状況を把握してから動きたいという、慎重さの表れでもある。

──────────────
C｜噂の出どころまで直接確かめに行ってしまう

行動力はある。ただし、勢いのまま確かめに行くと「そんなに気にしていたんだ」と、逆に噂に力を与えてしまうことがある。確かめること自体は間違いじゃない。動く前に一呼吸あるだけで、渡す情報の量が変わる。

──────────────
D｜時間が経てば「本人の言葉じゃない」と距離を置ける

これがいちばん消耗しない受け止め方だ。又聞きの言葉は、発した本人の言葉であって、あなたの真実じゃない。時間を置くと、それがちゃんと見分けられるようになる。

──────────────

本人に直接言わず、誰かを介して伝える。それは、正面から向き合う自信がない人の動きだ。動じずにいられる人ほど、噂はただの雑音として通り過ぎていく。
```
